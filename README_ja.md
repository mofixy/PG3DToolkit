**minecraft 用スキンを PG3D 用スキンへ加工、ゲーム内への一括インポートを可能にするウェブベースのツールです。**

- _現在は windows 専用です。_

## 使用方法

1. 一般的な Minecraft で使用されているスキンを入手する。
   - 64x64 サイズを推奨。64x32 サイズは検証していません。
2. https://mofixy.github.io/PG3DToolkit/ にアクセスする
3. ファイルの入力から入手したスキンをすべて選択する。
4. `generate`ボタンをクリックし、reg ファイルをダウンロードします。
5. 必要に応じてオプションを調整します。
6. .reg ファイルをダブルクリックして指示に従い、レジストリを変更します。
7. ゲーム内で確認してください。

## オプション

_すべてのオプションは`preview`ボタンを介してどのような変更が行われるか確認できます。_

- `Automatic skin processing`
  - スキンの自動加工を有効にします。このオプションをオフにすると 64x64 を 64x32 への変換、腕の太さを 3px から 4px への変換のみ実行されます。
- `Flip the top face of the arms on the X-axis`
  - スキンの肩に当たる部分の画像を反転させます。
  - PG3D の UV が奇妙なため、オフにすると影が外側へ向きます。
- `Remove dots that overwrite the eyeliner in the layer`
  - アイラインが上書きされるレイヤー側のドットを一部削除します。
  - 一般的なアバター型スキンではオンのほうが見た目が良いですが、一部のアバター型、特殊なスキンではオフを推奨します。
- `Skip overwriting all layer textures`
- `Skip overwriting the *** layer`
  - 全て、または一部のレイヤーの上書きをスキップします。
  - 一般的にオフのほうが見栄えが良くなりますが、一部のスキンでは非推奨です。

## 既知の問題

- 各種スキンを追加する形ではなく上書きする形でしかインポートできない
  - ⚠ すでにゲーム内でサーバー保存ではないスキンがある場合、それらは上書きされます。スキンの上書きが発生しても一切のサポートはできません！ ⚠
- スキンごとにオプションを設定できない
- 名前を設定できない
- `Automatic skin processing`オプションが完全に加工をオフにするものではない
- テクスチャを使用する左右の設定が機能していない

その他の問題が発生した際は issue にてご報告ください。
